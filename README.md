# helm-delete-all
A plugin for helm to delete all releases from all namespaces
## Installation
```
helm plugin install https://github.com/BarelElbaz/helm-delete-all
```
## Update
```
helm plugin update delete-all
```

## Usage:
```
helm delete-all
```
